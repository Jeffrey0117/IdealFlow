Connection Points

- 上方連接點 (in): 接收來自其他節點的連線
- 下方連接點 (out): 從此節點發出連線到其他節點
- 滑鼠從 out 拖曳到另一個節點的 in 即可建立連線

Drag & Drop

拖放功能用於分類模式和流程模式中的卡片整理。

互動說明

未分類
拖曳我到右邊
也可以拖曳我
已分類

使用方式

1. 分類模式: 在欄位之間拖曳卡片來改變分類
2. 流程模式: 從底部的未分配區域拖曳卡片到畫布上
3. 節點移動: 在流程模式中拖曳節點來重新排列位置
4. 建立連線: 從節點的輸出連接點拖曳到另一節點的輸入連接點

Data Actions

資料匯出與匯入功能按鈕。

功能說明

- 匯出 JSON: 將所有專案、想法、分類資料打包成 JSON 檔案下載
- 匯入 JSON: 上傳 JSON 備份檔案，可選擇合併或覆蓋現有資料
- 匯出檔案包含版本資訊和匯出時間戳
