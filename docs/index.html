Idea Flow

一個簡潔優雅的靈感管理系統，讓你隨時捕捉、整理、連結你的想法。 支援多專案管理、多種檢視模式，以及資料匯入匯出。

功能特色

Idea Flow 提供豐富的功能，幫助你更好地管理和組織靈感

📚

多專案管理

建立多個專案（腦袋），每個專案有獨立的人格圖示和色彩主題。 輕鬆在不同專案間切換，管理不同類型的想法。

📝

靈活的內容格式

使用 # 開頭的第一行作為標題，支援多行內容。 時間戳自動記錄，一目了然地查看靈感創建時間。

📁

分類整理

Kanban 風格的分類模式，可自訂分類並用拖放方式整理想法。 每個分類有獨特顏色，視覺化管理更清晰。

🔗

流程連結

在流程模式中建立想法之間的連結，用箭頭連接相關概念。 視覺化呈現思維脈絡，發現新的關聯。

💾

資料備份

支援 JSON 格式匯出與匯入，可選擇合併或覆蓋現有資料。 資料儲存在瀏覽器 localStorage，隨時備份不怕遺失。

📋

快速複製

一鍵複製想法內容到剪貼簿，方便分享或在其他應用中使用。 編輯、刪除操作都有確認機制，防止誤操作。

四種檢視模式

根據你的需求選擇最適合的檢視方式

列表模式

完整顯示每則想法的標題和內容，適合閱讀和瀏覽

卡片模式

網格排列的視覺化卡片，一目了然看到所有想法

分類模式

Kanban 看板風格，拖放卡片到不同分類欄位

流程模式

自由畫布配置節點，用箭頭連結想法建立思維圖

使用方法

幾個簡單步驟，開始記錄你的靈感

1

建立新專案

點擊「New Project」按鈕，輸入專案名稱並選擇一個人格圖示。 每個專案代表一個獨立的想法空間，有自己的色彩主題。

2

記錄想法

在輸入框中輸入你的想法，按 Post 或 Cmd/Ctrl + Enter 發布。 想要加標題？只需在第一行加上 # 符號。

# 這是標題
這是內容...
3

整理與分類

切換到分類模式，建立自訂分類，然後將想法拖放到對應的欄位中。 或使用流程模式，在畫布上連結相關的想法。

4

備份資料

回到專案選擇頁，點擊「匯出 JSON」備份所有資料。 需要時可以透過「匯入 JSON」還原或合併資料。
